Dustin and I worked together to create an hangman game in python. Dustin and I wrote and compiled this python program using an online software called repl.it. This software allowed us to edit and test our code in real time to discover if we had any issues/bugs with our code as we wrote (which we did).

A problem Dustin and I encountered while creating our python hangman game was that we did not know how to check for every letter before resetting guess. We knew about 'for' and 'while' commands, from prior python labs, which helped us fix this problem.We also ran into a problem crrating global varaibles, which are mandatory. We found out that we had to declare each of these variables as global prior to calling them in a sepearte function
