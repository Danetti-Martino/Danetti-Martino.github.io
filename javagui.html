Java Alarm GUI Project

Github Repo. For the Java Alarm GUI Proj.

Download link for the Java Alarm GUI Proj.


Abstraction is shown below:

The abstraction compares the integers for the hours, minutes, and seconds for the alarm, put in by the user, to the values returned by the 'cal.get(X)' functions, which return the actual time. Once the user integers of the alarm match the current time, the alarm will go off and a dialog box will appear.

The 'addalarmscreenlabels()' function is called upon the initialization of the alarm screen panel to add the labels to the panel.

The confirm button is clicked, then when the time is reached, a message dialog box appears showing a message, successfully alerting the user that the tm they inputted has been reached.
